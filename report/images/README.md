# Visualization Assets

This directory contains SVG illustrations for the Wikipedia vs Grokipedia quality comparison analysis.

## Available Visualizations

### 1. Overall Comparison (`overall-comparison.svg`)
- **Type:** Bar chart
- **Shows:** Average quality scores (26.7 vs 33.0 out of 35)
- **Highlights:** 94% vs 76% overall quality, +6.3 point advantage
- **Best for:** Executive summaries, presentations, social media

### 2. Key Findings (`key-findings.svg`)
- **Type:** Icon grid with statistics
- **Shows:** Six major findings in card format
- **Highlights:** Perfect accuracy tie, 94% quality, 7/7 topics won, timeliness, citations, depth
- **Best for:** Infographics, blog posts, newsletters

### 3. Dimension Comparison (`dimension-comparison.svg`)
- **Type:** Horizontal bar chart
- **Shows:** All 7 quality dimensions with scores
- **Highlights:** Accuracy tie, Grokipedia wins 6/7 dimensions
- **Best for:** Detailed analysis presentations, research papers

### 4. Topics Comparison (`topics-comparison.svg`)
- **Type:** Grouped bar chart
- **Shows:** Individual topic scores (out of 35)
- **Highlights:** Quality margins ranging from +2 to +11 points
- **Best for:** Topic-specific discussions, detailed comparisons

## Usage Guidelines

### File Format
All visualizations are in SVG format, which means they:
- Scale to any size without quality loss
- Can be embedded in HTML, Markdown, or presentations
- Have small file sizes (10-30KB each)
- Can be edited in vector graphics tools

### Embedding in Markdown
```markdown
![Overall Comparison](images/overall-comparison.svg)
```

### Embedding in HTML
```html
<img src="images/overall-comparison.svg" alt="Overall Comparison" width="800">
```

### Embedding in Presentations
Import directly into PowerPoint, Keynote, or Google Slides as vector graphics.

## Color Scheme

The visualizations use a consistent color palette:
- **Wikipedia Blue:** #2563eb (primary), #1e40af (dark)
- **Grokipedia Orange:** #f97316 (primary), #ea580c (dark)
- **Success Green:** #10b981
- **Warning Yellow:** #fbbf24
- **Gray Scale:** #1f2937 (dark), #6b7280 (medium), #e5e7eb (light)

## Customization

These SVG files can be customized by:
1. Opening in any text editor (they're XML)
2. Modifying colors, sizes, text
3. Using vector graphics software (Inkscape, Illustrator, Figma)
